A) Introduction - top

Soft Protractor is a useful professional tool to find the angle or level of a slope or an inclined plane.

Usage:

  1. This application is used to measures the angle and the slope of an object.
  2. Very helpful tool while fixing the things like photo frames, sceneries, lights, clocks, mirrors etc. on wall to keep them in level
  3. Professionals and engineers have found this application more convenient and applicable while working in field.

Features: - top

  1. Eclipse Editable
  2. Compatible with Android 2.2+
  3. Multiple Themes
  4. Beautiful Professional and Geometry glass protractor themes
  5. Special Feature included to measure the angle from back of the mobile device
  6. Three tilting units for measuring the angle like Degrees, Radians and Percentage, no need to conversion
  7. Digital Screens for displaying the measuring results like angle and pitch etc.
  8. Play/Pause button for noting the readings.
  9. Zero Calibration to fix the level error.
  10. User friendly interface

Accuracy:

The accuracy of the application is mainly dependent on your mobile device sensor efficiency. It is recommended to refresh your device sensor by moving it up and down diagonally in the figure of 8 before using this application

B) Getting Started - top

Before starting to use this source code and developing app you need to first install the following softwares to your computer. If you have already installed these softwares you can skip this step.

1. JDK 5 or above: Install Java Development Kit (JDK) 5 or greater. You can download here.



2. Eclipse IDE: You can choose Eclipse Indigo, Juno or other packages. Extract the zip file into a folder for example D:\ANDROID\Eclipse after downloading.

Run eclipse.exe On first time you will be asked for workspace directory. It is a directory where you want your android project to be stored. For now just create new folder Workspace in D:\ANDROID or any other directory as you wish.

3. Android SDK: Install android sdk. Now you need to connect Eclipse and Android sdk. On Eclipse GUI, select Windows > Preferences menu on top. Select Android and then locate your SDK Location, on my machine it is D:\ANDROID\android-sdk, and then click Apply.




C) Importing Project into Eclipse - top

  1. Extract the downloaded archive that you have purchased in CodeCanyone.
  2. Open Eclipse > Select File > Import.
  3. In the Import window page select Existing Projects into Workspace under General and click Next.
  4. In the next window choose Select archive file. Press the Browse button and choose Protractor.zip file present in the Project folder. Click Finish.
  5. Your project will be visible on the Package Explorer. At the left side you can see the project folder. You can edit your files from here.

D) Changing Package Name - top

You need to change package name in the .java files (src folder), and android manifest.

Replace in java files: Totally there are 9 .java files in src folder under main Package. Open all these .java files in Notepad++ and replace the package name with a package name of your choice in all files at once. E.g. soft.protractor.glass

Do not select "Match case". You can see 32 occurrences getting replaced.

AndroidManifest.xml: Replace the package name. Only one package definition is there at the top.

You may find it little hard to replace in first try, but in few attempts you can do it. For experienced users this is a simple task.
